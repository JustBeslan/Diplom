1  Программное обеспечение для входа пользователя в систему

Программа блокирующая доступ запрашивает с микрофона звуковой фрагмент и производит анализ и сравнение с сохраненными в базе данных тремя характеристиками голоса пользователя. Имя пользователя выбирается из заданного имени пользователя в системе. Рабочее окно программы на рисунке 1.

Рисунок 1 - Окно  программы блокировки доступа

Полученный спектр звукового фрагмента визуализируется на трехмерном графике.

2 Программное обеспечение для регистрации пользователя.

 

Программа состоит из диалогов:

- добавление пользователя;

- изменения параметров зарегистрированного в системе пользователя;

- настройки программы;

- настройки параметров работы администратора;

- просмотра статистики работы программы;

- удаление пользователя зарегистрированного в системе.

           

Основное окно показано на рисунке 2.

Рисунок 2 - Основное окно регистрации пользователей

Добавление пользователя

 

            В базу данных требуется добавить образцы голоса всех пользователей которые установлены в системе, или только тех пользователей для которых требуется голосовая защита. По умолчанию программа выбирает текущего пользователя, и если он всего один, то рекомендуется оставить этот логин. Имя Вы можете написать любое.

Вы должны записать 3 звуковых фрагмента своего голоса, выберете для этого одну не длинную фразу. На каждом из образцов есть зеленая кнопка, запись происходит когда она нажата.

Когда все 3 образца будут записаны нажмите кнопку сохранения.

3 Техническое обеспечение

 

Программа тестировалась на различных компьютерах класса не ниже Pentium 2. Компьютер необходимый для работы должен соответствовать требованиям WindowsXP.

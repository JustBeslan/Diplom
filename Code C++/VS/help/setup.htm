Установка

 

Для работы программы требуется установленный Interbase 5.х Client/Server. В нем требуется прописать пользователя EA c паролем ЕА. BDE для работы не требуется.

 

В установочном архиве находится 2 gdb файла, один пример с уже добавленным списком предприятий (>200 наименований) , другой чистый. Для программы рабочая база ea.gdb. Все необходимые настройки хранятся в ea.ini.

 

Пароль при запуске «1» - разрешает все, «2» - только чтение. Пароли хранятся в таблице Org_names, записи pass,pass1.

 

Внимание!

Подробное описание функций программы будет выслано только зарегистрированным пользователям. Им же гарантированна поддержка.
